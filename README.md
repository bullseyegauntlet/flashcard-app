# Flashcard App

A lightweight, offline-first flashcard app for studying.

## Features
- Create and manage decks
- Study mode with card flip animation
- Track learning progress
- All data stored locally (no accounts required)

## Getting Started
1. Clone the repo
2. Open `index.html` in your browser
3. Start creating decks!
