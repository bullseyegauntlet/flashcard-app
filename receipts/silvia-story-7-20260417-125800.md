**Pagination Strategy:**
- Display 50 cards per page (configurable via cardsPerPage)
- Deck with 1000 cards = 20 pages
- First page: cards 1-50
- Last page: cards 951-1000
- Navigation: Previous/Next buttons with disabled state

**Performance Optimizations:**
1. **Pagination:** Only 50 cards rendered at a time
2. **localStorage:** Already compact JSON
3. **Study mode:** Loads all cards in memory (acceptable for 1000 card decks)
4. **Deck View:** Slices cards on render (efficient)
