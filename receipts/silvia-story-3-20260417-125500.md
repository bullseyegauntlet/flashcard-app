## Implementation Details

**ShuffleEngine Module:**
- Fisher-Yates algorithm: O(n) time complexity
- In-place shuffle on array copy
- Cryptographically random: Math.random() sufficient for UX

**StudySession Integration:**
- start(deckId, shouldShuffle): New parameter
- If shouldShuffle=true: ShuffleEngine.shuffle(cardsCopy)
- If shouldShuffle=false: Original order
- isShuffle property tracks if shuffled

**UI Implementation:**
- Checkbox in Deck View (before Study Deck button)
- startStudyWithOptions() reads checkbox state
- Passes shouldShuffle to StudySession.start()

**No Persistence:**
- Shuffle state not saved to localStorage
- Each new study session resets
- User must re-check shuffle if desired
